## 1. 비전 & 철학

### 1.1 왜 질문인가

모든 위대한 사상가는 **답**이 아니라 **질문**으로 기억된다.

- 소크라테스: "선이란 무엇인가?"
- 데카르트: "확실하게 알 수 있는 것은 무엇인가?"
- 칸트: "우리는 무엇을 알 수 있는가?"
- 하이데거: "존재의 의미는 무엇인가?"
- 붓다: "고통의 원인은 무엇인가?"

질문은 시대와 문화를 초월하여 **변형되고, 분기하고, 합류**한다.
"존재란 무엇인가?"라는 질문은 파르메니데스에서 시작해 플라톤, 아리스토텔레스,
토마스 아퀴나스, 데카르트, 칸트, 하이데거를 거치며 2500년간 변형되었다.
동시에, 같은 질문이 인도에서는 나가르주나의 "공(空)", 중국에서는 노자의 "도(道)"로
독립적으로 탐구되었다.

### 1.2 질문 계보학

니체가 도덕의 '계보학'을 통해 "선/악 개념이 어디서 왔는가?"를 추적한 것처럼,
Sophia Atlas의 질문 계보 시스템은 "이 질문이 어디서 왔고, 어디로 갔는가?"를 추적한다.

```
┌─────────────────────────────────────────────────────┐
│                   질문 계보의 구조                      │
│                                                       │
│   [근원 질문]                                          │
│       │                                               │
│       ├── [변형 질문 A] ── 서양 경로                    │
│       │       ├── 답변: 플라톤 (이데아)                 │
│       │       ├── 답변: 아리스토텔레스 (형상)            │
│       │       └── 재변형: 칸트 (물자체)                 │
│       │                                               │
│       ├── [변형 질문 B] ── 불교 경로                    │
│       │       ├── 답변: 나가르주나 (공)                 │
│       │       └── 답변: 바수반두 (유식)                 │
│       │                                               │
│       └── [변형 질문 C] ── 도교 경로                    │
│               └── 답변: 노자 (도)                      │
│                                                       │
│   → 같은 근원 질문이 전통마다 다른 형태로 탐구됨          │
└─────────────────────────────────────────────────────┘
```

## 3. 데이터 모델

### 3.1 Question Interface

```typescript
interface Question {
  id: string;                          // "what-is-being", "what-is-knowledge"
  text: {
    ko: string;                        // "존재란 무엇인가?"
    en: string;                        // "What is Being?"
  };
  domain: QuestionDomain;              // 'metaphysics' | 'epistemology' | ...
  subdomain?: string;                  // '존재론' | '우주론' | ...
  keywords: string[];                  // ['존재', '실재', '본질']

  // 계층
  parentQuestion?: string;             // 상위 질문 id
  childQuestions?: string[];           // 하위 질문 id[]
  siblingQuestions?: string[];         // 같은 수준 관련 질문
  transformedFrom?: string;            // 변형된 원래 질문

  // 탐구 데이터
  era_first_asked?: Era;               // 최초 제기 시대
  explorers: QuestionExplorer[];       // 탐구자 목록
  traditions: TraditionPath[];         // 전통별 경로

  // 콘텐츠
  summary: string;                     // 질문 소개 (1~3문장)
  significance: string;                // 이 질문이 왜 중요한가
  contemporary_relevance?: string;     // 현대적 의미
}

interface QuestionExplorer {
  personId: string;                    // 인물 id
  position: string;                    // 해당 인물의 입장/답변 (1~2문장)
  approach?: string;                   // 접근 방법
  era: Era;
  year?: number;
  tradition: string;                   // 'western' | 'buddhist' | 'confucian' | ...
  keyWork?: string;                    // 관련 저작 id
}

interface TraditionPath {
  tradition: string;                   // 전통 이름
  path: {
    personId: string;
    year?: number;
    contribution: string;              // 이 전통에서의 기여 (1문장)
  }[];
  summary: string;                     // 이 전통의 탐구 경로 요약
}
```
