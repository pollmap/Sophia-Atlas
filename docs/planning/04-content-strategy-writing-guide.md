# Sophia Atlas — 콘텐츠 전략 & 집필 가이드 v1.0

> **"학술적 깊이 + 대중적 접근성 — 백과사전이 아닌 지적 여행 안내서"**
> 작성일: 2026-02-05 | 버전: 1.0 | 상태: Draft

## 1. 콘텐츠 비전

### 1.1 톤 & 보이스

Sophia Atlas의 콘텐츠는 **경외감과 호기심**을 동시에 불러일으킨다.
건조한 백과사전도 아니고, 가벼운 블로그도 아니다.
**교양 있는 안내자가 박물관을 안내하는 톤**이다.

```
톤 스펙트럼:

  학술 논문 ─── 백과사전 ─── [Sophia Atlas] ─── 교양 에세이 ─── 블로그
  (너무 딱딱)   (무미건조)   (여기)          (너무 가벼움)    (너무 캐주얼)
```

### 1.2 핵심 원칙

| 원칙 | 설명 | 예시 |
|------|------|------|
| **정확성** | 학술적으로 검증된 내용만 | "~라고 알려져 있다" (출처 필요) |
| **연결성** | 고립된 정보가 아닌 맥락 속 이해 | "칸트는 흄에 의해 '독단의 잠'에서 깨어났다" |
| **통찰** | 단순 나열이 아닌 의미 전달 | "왜 이것이 중요한가"를 항상 포함 |
| **균형** | 서양 편향 경계, 다양한 관점 | 동서양 비교, 비판적 시각 포함 |
| **접근성** | 전문가가 아닌 독자도 이해 | 전문용어에는 항상 짧은 설명 병기 |

## 2. 콘텐츠 유형별 가이드

### 2.1 요약 (Summary)

**목적**: 3초 안에 "이 인물이 누구인지" 파악.
**길이**: 1~3문장, 50~150자.
**공식**: `[핵심 정체성] + [가장 중요한 기여] + [역사적 맥락(선택)]`

**좋은 예시:**

| 인물 | 요약 |
|------|------|
| 소크라테스 | 서양 철학의 아버지. 문답법(산파술)을 통해 진리를 탐구했으며, 직접 저술을 남기지 않았으나 제자 플라톤을 통해 그의 사상이 전해진다. |
| 다윈 | 자연선택에 의한 진화론을 제시하여 생물학과 인류의 자기 이해를 근본적으로 변혁시킨 영국의 박물학자. |
| 공자 | 동아시아 사상의 토대를 놓은 중국 춘추시대의 철학자. 인(仁)과 예(禮)를 핵심으로 하는 도덕 정치 사상을 제시했다. |
| 뉴턴 | 만유인력의 법칙과 운동 법칙으로 고전역학을 체계화하고, 미적분을 독립적으로 발명한 근대 과학혁명의 완성자. |
| 루미 | 페르시아 수피즘의 대표적 시인이자 신비가. 사랑을 통한 신과의 합일을 노래한 '마스나비'의 저자. |

**나쁜 예시 (피해야 할 것):**

```
❌ "유명한 철학자이다." — 너무 모호
❌ "기원전 470년에 태어나 399년에 사망했다." — 생몰년은 요약이 아님
❌ "소크라테스, 플라톤, 아리스토텔레스는..." — 다른 인물 중심 서술
❌ "매우 위대하고 영향력 있는..." — 수식어 남발
```

### 2.2 상세 설명 (Detailed)

**목적**: 인물/엔터티의 심층 이해.
**길이**: 500~2000자 (보통 800~1200자).
**구조**:

```
1단락: 도입 (핵심 정체성 + 시대적 배경)
2단락: 생애/경력 (핵심 사건 중심)
3단락: 핵심 사상/업적 (가장 중요한 내용)
4단락: 영향력 (누구에게, 어떻게)
5단락: 현대적 의미 (왜 지금도 중요한가)
```

**예시: 소크라테스 상세 설명**

```
소크라테스(기원전 470년경~399년)는 아테네에서 태어난 고대 그리스의 철학자로,
서양 철학 전통의 근본적 전환점을 만든 인물이다. 그 이전의 철학자들이
자연의 근본 원리(아르케)를 탐구한 것과 달리, 소크라테스는 인간 자신—
덕, 지식, 정의, 아름다움—을 철학의 중심에 놓았다.

그의 가장 독특한 방법은 '산파술(elenchus)'이라 불리는 문답법이었다.
소크라테스는 대화 상대방에게 질문을 던져 그들 스스로의 무지를 자각하게 했다.
그 자신도 "나는 내가 모른다는 것을 안다(무지의 자각)"고 고백했는데,
이는 단순한 겸손이 아니라 진정한 탐구의 출발점이었다.

소크라테스는 아테네 민주정에 의해 "청년을 타락시키고 새로운 신을 도입했다"는
죄목으로 사형을 선고받았다. 탈출할 기회가 있었으나, 법을 존중하는 원칙에 따라
독배를 마시고 죽음을 택했다. 이 장면은 플라톤의 『파이돈』에 생생히 기록되어 있다.

소크라테스의 직접적 영향은 제자 플라톤을 통해 서양 철학 전체로 퍼져나갔다.
플라톤의 이데아론, 아리스토텔레스의 형이상학, 스토아학파의 윤리학,
심지어 기독교 신학에 이르기까지, 소크라테스의 그림자는 서양 사상 전체에 드리워져 있다.

오늘날에도 "소크라테스식 대화법"은 교육학, 법학, 심리치료에서 활용되며,
"검토되지 않은 삶은 살 가치가 없다"는 그의 선언은 비판적 사고의 영원한 표어로 남아 있다.
```

### 2.3 관계 설명 (Relationship Description)

**공식**: `[관계의 성격] + [구체적 내용] + [역사적 의미]`
**길이**: 1~2문장.

| 관계 | 설명 |
|------|------|
| 소크라테스 → 플라톤 (teacher_student) | 소크라테스는 플라톤의 스승으로, 대화법과 산파술은 플라톤 철학의 출발점이 되었다 |
| 흄 → 칸트 (influenced) | 흄의 경험론적 회의주의는 칸트를 '독단의 잠'에서 깨워 비판철학의 탄생을 이끌었다 |
| 마르크스 ↔ 엥겔스 (collaborated) | 마르크스와 엥겔스는 공산당 선언과 자본론 집필에서 긴밀하게 협력한 평생의 동지였다 |
| 뉴턴 → 과학혁명 (participated) | 뉴턴의 프린키피아(1687)는 과학혁명의 완성을 상징하며, 기계론적 세계관을 확립했다 |

### 2.4 명언 (Quotes)

**선정 기준:**
1. 인물의 핵심 사상을 대표하는 것
2. 대중에게 영감을 줄 수 있는 것
3. 출처가 확인 가능한 것 (위작/오인용 배제)

**형식:**

```json
{
  "text": "검토되지 않은 삶은 살 가치가 없다.",
  "source": "플라톤, 『변론』 38a",
  "year": -399,
  "context": "재판에서 사형 선고를 받은 후 발언"
}
```

**번역 원칙:**
- 의미 전달 우선 (직역보다 의역)
- 원문 병기 권장 (특히 유명 구절)
- 여러 번역이 있을 경우 가장 널리 알려진 것 채택

## 3. 카테고리별 콘텐츠 가이드

### 3.1 철학자

**필수 다룰 항목:**
- 소속 학파 / 사상적 전통
- 핵심 개념 (2~5개)
- 주요 저작 (1~5개)
- 탐구한 철학적 질문
- 반대/계승 관계

**시대별 서술 특수사항:**

| 시대 | 특수사항 |
|------|----------|
| 고대 | 단편/구전 중심, 후대 전승 주의, 신화적 맥락 |
| 중세 | 신학과의 관계, 아랍어 번역 경로, 대학 제도 |
| 근대 | 과학혁명 맥락, 국민국가 형성, 인쇄술의 영향 |
| 현대 | 분과 전문화, 학파 간 논쟁, 현실 정치와의 관계 |

### 3.2 종교 인물

**필수 다룰 항목:**
- 소속 종교 + 역할 (예언자, 성인, 교부, 사제, 승려)
- 핵심 가르침
- 경전과의 관계
- 분파/전통에의 영향

**종교적 민감성 처리 원칙:**
1. **학술적 객관성**: "~라고 가르쳤다" (신앙 고백이 아님)
2. **비교종교학적 관점**: 판단 없이 서술
3. **내부자 관점 존중**: 신자의 이해 방식도 소개
4. **논쟁적 주제**: 학술적 다수설 + 이견 병기

### 3.3 과학자

**필수 다룰 항목:**
- 연구 분야
- 핵심 발견/이론 (평이한 설명 + 전문 설명)
- 실험/관측 방법
- 선행 연구와의 관계
- 과학사적 의의

**과학 용어 설명 원칙:**
```
전문용어 (평이한 설명)
예: "특수 상대성이론(빛의 속도에 가까워질수록 시간이 느려진다는 이론)"
예: "자연선택(환경에 더 잘 적응한 개체가 더 많이 생존하고 번식하는 과정)"
```

### 3.4 역사 인물

**필수 다룰 항목:**
- 시대적 배경
- 핵심 업적/사건
- 정치적 맥락
- 역사적 평가 (긍정/부정 양면)

### 3.5 문화 인물

**필수 다룰 항목:**
- 예술 분야 + 양식/사조
- 대표작 (3~5개)
- 혁신점 (무엇이 새로웠는가)
- 후대에의 영향

## 4. 엔터티 콘텐츠 가이드

### 4.1 사건 (Event)

**구조**: 배경 → 전개 → 결과 → 역사적 의의
**예시 (프랑스 혁명):**
```
배경: 구체제(앙시앵 레짐)의 모순과 계몽주의 사상의 확산
전개: 바스티유 습격(1789) → 인권선언 → 공화정 → 공포정치 → 나폴레옹
결과: 절대왕정 붕괴, 근대 민주주의의 출발
의의: 계몽주의 사상이 현실 정치로 전환된 결정적 사건
```

### 4.2 사상/이념 (Ideology)

**구조**: 정의 → 기원 → 핵심 원리 → 주요 사상가 → 역사적 영향

### 4.3 개념 (Concept)

**구조**: 정의 → 최초 제안자 → 역사적 변형 → 현대적 의미
**예시 (이데아):**
```
이데아(Idea)는 플라톤이 제시한 형이상학적 개념으로,
감각으로 인식되는 현상계 너머에 존재하는 영원불변의 본질을 뜻한다.
플라톤은 동굴의 비유를 통해, 우리가 보는 세계는 이데아의 그림자에
불과하다고 설명했다. 아리스토텔레스는 이데아가 개별 사물 안에
내재한다고 비판하며 형상(eidos) 개념으로 대체했다.
이데아론은 이후 신플라톤주의, 기독교 신학, 근대 관념론에
지대한 영향을 미쳤다.
```


## 7. 문체 가이드

### 7.1 기본 규칙

- **경어체**: 합쇼체 사용 ("~이다", "~한다")
- **문장 길이**: 평균 20~40자, 최대 60자
- **단락**: 1단락 = 1아이디어, 3~5문장
- **능동태 선호**: "칸트가 주장했다" > "칸트에 의해 주장되었다"

### 7.2 피해야 할 표현

```
❌ "매우 위대한", "전무후무한", "가장 뛰어난" → 과도한 수식
❌ "누구나 알다시피", "당연히" → 전제 강요
❌ "흥미롭게도", "놀랍게도" → 불필요한 감정 유도
❌ "~적인 것으로 보여지는" → 불필요한 피동 + 중복
✅ "~라고 주장했다", "~를 제시했다", "~에 기여했다" → 객관적 서술
```

## 8. 콘텐츠 생산 파이프라인

### 8.1 작성 프로세스

```
1차: 데이터 수집
├── 학술 자료 참조 (SEP, Britannica, 학술DB)
├── 기본 정보 확인 (생몰년, 지역, 카테고리)
└── 관계 맵핑

2차: 초안 작성
├── AI 보조 초안 생성 (프롬프트 가이드 활용)
├── 요약 + 상세 + 명언 + 저작
└── 교차 참조 링크 삽입

3차: 검수
├── 팩트체크 (날짜, 이름, 인용)
├── 문체 일관성 확인
└── 교차 참조 정합성 확인

4차: 최종 승인
├── 도메인 전문가 리뷰 (선택)
└── 데이터 검증 스크립트 통과
```

### 8.2 MVP 200명 작성 우선순위

**Tier 1 (필수, 50명)**: 모든 교양인이 아는 인물
```
소크라테스, 플라톤, 아리스토텔레스, 데카르트, 칸트, 니체,
마르크스, 붓다, 예수, 무함마드, 공자, 노자,
다윈, 뉴턴, 아인슈타인, 갈릴레오,
알렉산드로스, 나폴레옹, 간디, 링컨,
셰익스피어, 다 빈치, 바흐, 베토벤, 호메로스...
```

**Tier 2 (중요, 80명)**: 해당 분야를 이해하려면 반드시 알아야 할 인물

**Tier 3 (보완, 70명)**: 관계 네트워크 밀도를 위해 필요한 인물
